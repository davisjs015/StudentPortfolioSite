naniZoo

Brief

The goal of this project was to create kid-friendly drag and drop puzzle game while giving the user some general information about the puzzle animal once they puzzle is complete. The goal of the project was to create a drag and drop puzzle game for children during Fall 2017's WNM 617: Mobile Web Technology 1 class under Instructor Matthew David. Overall the focus was on making the puzzles child friendly with a surprise factoid or two after it is solved. This makes the game more interactive and educational. The game is fun for all ages but the primary audience is meant to be children under 10 years old with a secondary market around 5 years old.

Typeface

Aa

The typeface chosen for naniZoo was Oxygen that was designed by Vernon Adams as part of the KDE project. The decision to use Oxygen was based on the fact it is great on mobile devices and graphic designs. The typeface is a sans serif but encompasses features from the the different sans serif's to create a familiarity but yet newness combined with legibility. This made it ideal, as it's new but yet familiar, making it a a perfect combination for a children’s game.

Technology

The wireframes were created with pen and paper before being used as a guide for the animals, puzzles and prototypes created in Adobe Illustrator. The high-fidelity prototypes were created in Adobe XD and viewed within XD’s app to get a true feel for the game on its intended screen size. The mockups were created using Adobes iPhone 6/7 templates within Adobe Illustrator.

App

The actual HTML pages were created using the multi-page jQuery layout. This allowed more fluidity and control over the coding process as everything was all on one screen. It also made the transition from page to page smoother and easier to transfer when the user wants to navigate the app.
