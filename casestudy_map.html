Climate Change

Brief

Design a user friendly interactive infographic that describes the effects of climate change on sea temperatures and levels. The goal of this project was to design an interactive infographic for Fall 2017's WNM 661: Interactive InfoGraphics under INstructor Frederick McHale and make it into a fully integrated prototype in Adobe After Effects to show the user how the infographic would work if fully coded.

Typeface

Aa

The typeface used for the infographic was Avenir Next that was designed by Adrian Frutiger with the help of Linotype’s designer Akita Kobayashi. The decision to use Avenir Next was based on the large x-heights, multiple font weights, and legibility. In order to create a user-friendly appeal to the viewers the typeface had to have a warm and inviting feel but be legible.

Technology

The rough, refined, and final storyboards were created in Adobe Illustrator. The files were then imported to Adobe After Effects before animation was added to show how they would function if fully coded. Desktop mockups were created in Adobe Illustrator using Adobe’s desktop templates.
